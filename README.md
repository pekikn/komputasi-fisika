# Komputasi Fisika

Repository ini merupakan kode-sumber (*source-code*) dari modul **Komputasi Fisika**.

## Template Pembuatan Modul

Untuk memanfaatkan sebagai template pembuatan Modul yang sejenis dengan modul **Komputasi Fisika** maka gunakan perintah berikut: `git clone https://github.com/pekikn/komputasi-fisika`

## Cara Pembuatan Modul 

Informasi untuk pembuatan model dapat mengikuti panduan seperti yang diberikan pada  **Jupyter Book versi 2** atau **MyST Markdown**.
